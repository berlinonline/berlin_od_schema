Berlin Open Data Schema

Schema to describe and validate dataset descriptions for the Berlin Open Data Portal at http://daten.berlin.de. Using JSON Schema v4. The base schema is the CKAN dataset schema. It's interpretation and the extras are similar to the OGD (Open Government Deutschland) schema defined here: https://github.com/fraunhoferfokus/ogd-metadata.

Uses http://json-schema.org/draft-04/schema#
